# ENChatter
ENChatter is a chat client for Twitch chat

*UNDER DEVELOPMENT*

### Installation - Clone repo
```
git clone https://github.com/encoderpie/enchatter.git
cd enchatter
npm i
```

### Run Electron - Desktop app
```
npm start
```
